npm install commands

2016-11-17

npm install -g [name] install globally

npm install --save [name] this name will be installed locally and shown in the package.json under the attribute: depenencies

npm install --save-dev [name] installed locally and shown under devDependencies

After deleting the ‘node_modules’ folder, just run npm install, it will install the above two
