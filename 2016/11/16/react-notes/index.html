There are two types of data in a component:

  • props: get passed into a component as initialized; for example
    ReactDom.render(<MyList name="Kevin"/>, document.getElementById('app'))
    A component cannot update the props, all you can do is set the default value in getDefaultProps function in the component.
  • state: internally maintained and updated by the component.

Two ways of constructing a component

  1. using var myComponent = React.createClass({ ... }) example
  2. using class MyCompoent extends React.Component{ ... } example
    Note: No Autobinding Methods follow the same semantics as regular ES6 classes, meaning that they don’t automatically bind this to the instance. You’ll have to explicitly use .bind(this) or arrow functions. See here.
